# Basic-Tools

##### A self-hosted website that offers useful digital Tools
##### 

## Installation

1. Clone this repository

2. Install client modules
```bash
cd client
npm install
```

3. Move up 1 directory
```bash
cd ..
```

4. Install server modules
```bash
cd server
npm install
```
